# Ways to find the Server's Real/Origin IP address:

## Passive
 - DNS Records
 - CIDRs
 - SSL Certificates
 - Favicon Hashes
 - Shodan, Censys etc.


## Active
- XML/RPC Pingbacks
- Virtual Hosts
- Google Analytics
- Re-captcha Identifier Keys
- Custom Headers
